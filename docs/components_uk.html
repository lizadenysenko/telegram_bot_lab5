Взаємодія компонентів

Основні компоненти:

  1. Користувач — надсилає команду боту.
  2. Telegram Bot API — пересилає повідомлення до логіки на бекенді у bot.py.
  3. OMDb API — виконує пошукові запити за параметрами (наприклад, назва фільму).
  4. Бот — отримує відповіді у форматі JSON, обробляє їх і формує повідомлення для користувача.

Уся логіка ґрунтується на обробниках команд (@bot.message_handler).
