Архітектура проєкту

Бот побудований за модульною структурою:

  • bot.py — головний файл з логікою Telegram-бота.
  • Використовується OMDb API для отримання інформації про фільми.
  • Telegram-бот обробляє команди користувача, аналізує введення та надсилає відповіді.
  • Функції ізольовані та структуровані за відповідальністю: розбір введення, API-запити, форматування повідомлень.

Використані технології:

  • Python 3.12
  • Telegram Bot API (pyTelegramBotAPI)
  • OMDb API
  • Sphinx (для документації)
